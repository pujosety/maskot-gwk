# maskot-gwk

> A playful mascot character toolkit and mini game starter for quick, game-like experiences.

`maskot-gwk` is an open-source project born from the **GWK** creative space. It gives you a lightweight, friendly starting point for building character-driven apps, playful interactions, and mini games — without the bloat of a full game engine.

---

## Why this exists

Most game-ready repos assume you are making a big platformer or RPG. If you actually need something smaller — like a mascot opener, an onboarding game, a campaign character interaction, or a playful animated tool — existing templates are usually too heavy and hard to extend.

This repo changes that.

---

## What you get

- Mascot/character-focused project setup
- Simple animation and interaction loops
- Easy-to-scan structure
- Friendly for custom branding and events

---

## Perfect for

- Freelancers building character-based interactions
- Startups validating playful concepts quickly
- Students learning 2D animation basics
- Open-source contributors who love clean, compact apps

---

## Tech Stack

- Flutter / Dart-friendly setup
- Lightweight rendering approach
- Easy asset integration

---

## Getting Started

1. Clone the repository.
2. Open the project in your IDE.
3. Replace starter assets with your own mascot and visuals.
4. Run the app to preview interactions.
